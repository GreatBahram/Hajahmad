## Want to develop on TripAdvisor?

This gets you started:

```bash
$ git clone https://github.com/GreatBahram/TripAdvisor.git
$ cd TripAdvisor
$ virtualenv venv
$ . venv/bin/activate
$ pip3 install -r requirements.txt 
$ python3 run.py -h
```
